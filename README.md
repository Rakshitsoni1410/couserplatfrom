# 🎓 NextSkills – Online Course Platform

NextSkills is a full-stack online learning platform built as my **BCA Final Year Project**, inspired by platforms like **Udemy** and **Coursera**.

It allows students to explore and purchase courses, watch lectures, track learning progress, and interact with instructors through reviews. Instructors can create and manage courses, upload lectures, and respond to student feedback.
